# sales-performance-dashboard
📊 Sales Performance Dashboard | Power BI

An interactive Sales Performance Dashboard built using Microsoft Power BI to analyze sales, profit, quantity, customer segments, product categories, and regional performance through dynamic visualizations and KPI metrics.

📌 Project Overview

This project provides a comprehensive analysis of business sales performance using Power BI. The dashboard transforms raw sales data into meaningful business insights, enabling users to monitor sales trends, profitability, product performance, customer segments, and regional sales through interactive visualizations.

The project demonstrates data cleaning, data modeling, DAX calculations, KPI creation, and dashboard design skills required for Data Analyst and Business Intelligence roles.

🎯 Objectives
Analyze overall sales performance.
Track monthly sales and profit trends.
Compare sales across customer segments.
Identify top-performing products and categories.
Analyze regional and state-wise sales.
Build an interactive business intelligence dashboard.
